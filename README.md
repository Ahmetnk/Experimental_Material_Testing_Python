Mechanical Materials Laboratory Data Analysis

This repository contains Python scripts developed for laboratory experiments in Mechanics and Materials II (ME 232) during my undergraduate studies in Mechanical Engineering.

The purpose of these scripts was to process experimental data, perform engineering calculations, generate plots, and support the analysis presented in the laboratory reports.

Projects
1. Tensile Test Analysis

Analysis of metallic and polymeric tensile test data.

Main tasks:

Importing stress-strain data from Excel files
Plotting stress-strain curves
Calculating Young's Modulus
Determining Ultimate Tensile Strength (UTS)
Comparing results between experimental groups
2. Hardness Test Analysis

Analysis of Vickers and Shore hardness measurements.

Main tasks:

Processing hardness measurements from multiple specimens
Calculating mean values and standard deviations
Estimating mechanical properties from hardness data
Investigating the relationship between Shore Hardness and Young's Modulus
Data visualization
3. Fracture Mechanics Analysis

Evaluation of fracture toughness from experimental measurements.

Main tasks:

Reading experimental fracture data
Calculating fracture toughness (KIC)
Statistical analysis of experimental results
Visualization and comparison of specimen behavior
Technologies Used
Python

Purpose

These scripts were created as educational engineering tools to support laboratory experiments and material characterization studies. The focus was on data analysis and engineering calculations rather than software development practices.
